  • By far the lengthiest part of writing this function was utilizing the “dictionaries” provided by the Census Bureau and converting them to named lists. I am sure there were more elegant methods than what I employed, but it came down to using Find + Replace commands numerous times and then just manually editing fields (such as the average times per code) by hand.

  • Also time consuming were the checks for the data being entered correctly. I would add that all the examples provided by the Census Bureau include &SCHL=24 within the query. Removing or altering this portion results in an error, so note that the data will ALWAYS include SCHL (fixed at 24 [Doctoral Degree]), regardless of the user’s query. If I had more time, I’d include another argument for different schooling levels.

  • The part I most enjoyed about this was actually converting the data to numeric and quantitative with the (cunning) use of if checks.

  for (i in 2:length(years)){
    census_working <- census_query(year = years[i],
                                   numvar = numvar,
                                   catvar = catvar,
                                   geolevel = geolevel,
                                   subset = subset)
    census_working <- add_column(census, "year" = years[i], .before = 1)
    
    census_main <- bind_rows(census_main, census_working)
  }
  
  census_main
